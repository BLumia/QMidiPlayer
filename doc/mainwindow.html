Main Window


  1. Now playing. Right clicking here will bring up the action menu.
  2. Seeking slider. Song length is estimated and may not be 100% accurate.
  3. Playback control buttons. Self-explanatory.
  4. Open file.
  5. Settings button. Refer to the Settings section for details.
  6. Dialog buttons. You can configure this section in settings.
  7. FluidSynth master gain slider.
  8. FluidSynth polyphony indicator.
  9. A dumb button.

File action menu

Note that actual actions in this menu can be configured in settings.

  • File Information.
    Click an item in this dialog to copy its content.
  • Render to Wave.
    It's not possible to use this action unless the player is stopped.
    The output wave file, with the name (source file name)+'.wav',
    is always placed in the folder that the source midi file is in.
    MIDI mapping does not apply to render results.
  • Panic.
    If somehow the player messed up, use this action to turn off all the notes.
  • Restart fluidsynth.
    This action can only be used when no file is being played.
    Applies settings and restart the internel fluidsynth engine.
