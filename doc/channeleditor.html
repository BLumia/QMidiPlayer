Channel Editor


Controllers Explained

  • Cut.: Cut Off (cc#74, defined in GM2).
  • Res.: Resonance (cc#71, defined in GM2).
  • Rev.: Reverb (cc#91, defined in GM1).
  • Chr.: Chorus (cc#93, defined in GM1).
  • Vol.: Volume (cc#07, defined in GM1).
  • Pan.: Panorama (cc#10, defined in GM1).
  • Atk.: Envelope attack (cc#73, defined in GM2).
  • Dec.: Envelope decay (cc#75, defined in GM2).
  • Rel.: Envelope release (cc#72, defined in GM2).
  • Rate: Vibrato rate (cc#76, defined in GM2).
  • Dep.: Vibrato depth (cc#77, defined in GM2).
  • Del.: Vibrato delay (cc#78, defined in GM2).

Notes

  • Changes in this page are applied immediately.
  • If you are using fluidsynth, the actual effect of a control change depends on the soundfont you use.
