Channel Editor


Controllers Explained

  • Cut.: Cut Off (cc#74, defined in GM2). Not supported by vanilla fluidsynth.
  • Res.: Resonance (cc#71, defined in GM2). Not supported by vanilla fluidsynth
  • Rev.: Reverb (cc#91, defined in GM1).
  • Chr.: Chorus (cc#93, defined in GM1).
  • Vol.: Volume (cc#07, defined in GM1).
  • Pan.: Panorama (cc#10, defined in GM1).
  • Atk.: Envelope attack (cc#73, defined in GM2). Not supported by vanilla fluidsynth.
  • Dec.: Envelope decay (cc#75, defined in GM2). Not supported by vanilla fluidsynth.
  • Rel.: Envelope release (cc#72, defined in GM2). Not supported by vanilla fluidsynth.
  • Rate: Vibrato rate (cc#76, defined in GM2). Not supported by vanilla fluidsynth.
  • Dep.: Vibrato depth (cc#77, defined in GM2). Not supported by vanilla fluidsynth.
  • Del.: Vibrato delay (cc#78, defined in GM2). Not supported by vanilla fluidsynth.

Notes

  • Changes in this page are applied immediately.
  • Vanilla fluidsynth does not support GM2 controls.
